# 量化压缩
神经网络中的浮点数（FP32/FP16）转换为低位整数（INT8）
**量化公式**：  
INT8_value = round(FP_value / scale) + zero_point  
**反量化公式**：  
FP_value = scale × (INT8_value - zero_point)
**参数说明**：  
- `scale`（浮点数）：缩放因子，定义了INT8的"1"等于多少浮点值
- `zero_point`（整数）：零点偏移，确保浮点0.0能准确映射到INT8范围内
